Stacks

만들고자 하는 목표에 대해 필요한 기술들을 빠르게 습득하고 구현합니다. 할 수 있는 경계선을 더 넓혀 나가고자 합니다.
기술 구분 로고 활용 수준 평가
Java Spring MVC Framework를 기반으로 개발하며 다양한 웹 분야에도 능숙히 개발해낼 수 있습니다.
주로 백 엔드(Back-end)와 풀 스택(Full Stack)에 비중을 크게 두고 있습니다.
JavaScript 프론트엔드의 주 언어로 JavaScript를 사용하고 있으며, 각종 라이브러리를 활용할 수 있습니다.
주로 JQuery를 활용하며 Node.js를 사용해본 경험이 있습니다.
MySQL 관계형 데이터베이스를 설계하고 MySQL을 사용하여 높은 생산성으로 웹 서비스를 개발하고 있습니다
MySQL의 프론트엔드인 Heidi SQL을 사용하여 관리하고 있습니다.

Front End

JavaScript를 가장 많이 사용하고 있습니다. 추가적으로 JavaScript의 라이브러리는 JQuery를 사용하고 있습니다.

Back end

기본 언어는Java를 사용하며 Spring 프레임워크를 주로 활용하고 있습니다. 추가적으로 Python를 공부해보고 싶습니다.

Data base

MySQL을 주로 사용하고 있습니다. 추가적으로 Oracle을 사용해본 경험이 있고, NoSQL 중 하나인 MongoDB에 관심이 많습니다.
